¶基础概念

S[i]S[i]S[i]：表示字符串 SSS 第 iii 个位置的字母，下标从 111 开始。
PrefixS[i]PrefixS[i]PrefixS[i]：表示字符串 SSS 的长度为 iii 的前缀，PrefixS[i]=S[1,i]PrefixS[i] = S[1, i]PrefixS[i]=S[1,i]。
SuffixS[i]SuffixS[i]SuffixS[i]：表示字符串 SSS 的长度为 iii 的后缀，SuffixS[i]=S[∣S∣−i+1,∣S∣]SuffixS[i] = S[|S| − i + 1, |S|]SuffixS[i]=S[∣S∣−i+1,∣S∣]。
BorderBorderBorder：如果字符串 SSS 的同长度的前缀和后缀完全相同，即 Prefix[i]=Suffix[i]Prefix[i] = Suffix[i]Prefix[i]=Suffix[i]，则称此前缀（后缀）为一个BorderBorderBorder。
特殊地，字符串本身也可以是它的 BorderBorderBorder。
周期:对于字符串 $S4 和正整数 $p4，如果有 S[i]=S[i−p]S[i] = S[i − p]S[i]=S[i−p]，对于 p<i≤∣S∣p < i ≤ |S|p<i≤∣S∣ 成立，则称 ppp 为字符串 SSS 的一个周期。
特殊地，p=∣S∣p = |S|p=∣S∣ 一定是 SSS 的周期。
循环节：若字符串 SSS 的周期 ppp 满足 p∣∣S∣p | |S|p∣∣S∣，则称 ppp 为 SSS 的一个循环节。
特殊地，p=∣S∣p = |S|p=∣S∣ 一定是 SSS 的循环节。

¶重要性质

ppp 是 SSS 的周期 ⇔ ∣S∣−p|S| − p∣S∣−p 是 SSS 的 BorderBorderBorder
传递性：SSS 的 BorderBorderBorder 的 BorderBorderBorder 也是 SSS 的 BorderBorderBorder。
求 SSS 的所有 BorderBorderBorder 等价于求所有前缀的最大 BorderBorderBorder。
BorderBorderBorder 的性质
周期定理：若 p,qp, qp,q 均为串 SSS 的周期，则 (p,q)(p, q)(p,q) 也为 SSS 的周期。
一个串的 BorderBorderBorder 数量是 O(N)O(N)O(N) 个，但他们组成了 O(logN)O(logN)O(logN) 个等差数列。

¶AC自动机

ACACAC 自动机 = TrieTrieTrie + KMPKMPKMP
ACACAC 自动机基于 TrieTrieTrie，将 KMPKMPKMP 的 BorderBorderBorder 概念推广到多模式串上。
ACACAC 自动机是一种离线型数据结构，即不支持增量添加新的字符串。
ACACAC 自动机常用于将字符串询问类的问题进行离线处理，也经常与各种DPDPDP 结合，或是补全成 TrieTrieTrie 图。
广义 borderborderborder:
推广到两个串：对于两个串 SSS 和 TTT ，相等的 ppp 长度的 SSS 的后缀和 TTT 的前缀称为一个 borderborderborder。
推广到一个字典：对于串 SSS 和一个字典 DDD，相等的 ppp 长度的 SSS 的后缀，和任意一个字典串 TTT 的前缀称为一个 borderborderborder。
失配（FailFailFail）指针：
对于 TrieTrieTrie 中的每一个节点（即某个字典串的前缀），它与 TrieTrieTrie 中所有串的最大 BorderBorderBorder 即为失配指针。
类似与 KMPKMPKMP 求 BorderBorderBorder，任意节点的 BorderBorderBorder 长度减一，一定是父节点的BorderBorderBorder。
因此可以通过遍历父节点的失配指针链来求解。
因此在求失配指针的时候，一定要按长度从小到大来求，即 bfsbfsbfs。
复杂度分析:
类似于 KMPKMPKMP 的势能分析方法，势能总量等于 TrieTrieTrie 的节点总数，因此复杂度为线性的。
